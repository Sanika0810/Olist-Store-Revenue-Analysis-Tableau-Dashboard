## ** Project Overview**

This project presents an interactive **Tableau dashboard** built using the Olist Brazilian E-commerce dataset.

## **Project Workflow**

1. Imported all 9 Olist datasets into Tableau
2. Performed data cleaning (data types, joins, category mapping)
3. Built calculated fields (AOV, review averages, revenue metrics)
4. Designed dashboard layout with filters & KPIs
5. Created interactive charts for revenue and order analysis
